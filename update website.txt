This is a step by step of commands to type in (git bash) when the "update website" button is not working
(pushes everything to live update and in correct folders)
--------------------------------------------------
>>>Open Git Bash

>>>Go to your repo directory, paste below line and press enter:
cd /c/Users/(name)/Desktop/(website)


>>>Get remote changes first (sync), paste below line and press enter:
git pull origin main --rebase


>>>Upload (push) your changes, paste below line and press enter:
git push origin main
